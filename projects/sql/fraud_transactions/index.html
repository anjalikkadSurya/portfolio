Fraud Chain Detection

This project analyzes a dataset of cryptocurrency transactions to identify suspicious chains of transfers within tight time windows.

  • Goal: Find consecutive transactions within 5 minutes of each other.
  • Criteria: Total amount > 150 in each chain.
  • Output: Rank chains by total amount lost per sender.

Expected Output

The output contains the following fields:

  • sender_id
  • chain_id (unique per sender)
  • chain_start and chain_end timestamps
  • total_amount, num_txns, fraud_rank
  • transaction_ids (involved in chain)
